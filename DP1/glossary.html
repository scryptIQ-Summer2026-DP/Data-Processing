Data Processing 1 - Glossary

Glossary DP1
NaN (Not a Number): A special float value used to represent missing or undefined data; most NumPy and Pandas functions handle NaN values explicitly, either ignoring or propagating them.
NumPy: A core Python package for numerical computing that provides the `ndarray` — a fast, memory-efficient array structure — along with a comprehensive suite of mathematical functions.
NumPy array (ndarray): The central data structure of NumPy — a homogeneous, n-dimensional array of numerical values that supports vectorised operations and broadcasting.
OR (|) operator: Returns `True` when at least one condition is `True`; used in boolean masking to combine multiple conditions where any one being satisfied is sufficient.
Pandas: An open-source Python package built on top of NumPy that provides the DataFrame and Series structures for handling, augmenting, and analysing tabular data.
Path: The address of a file or folder on a computer's filesystem, expressed as a sequence of directory names.
pip: The standard Python package installer, used to install third-party packages from the command line (e.g. `pip install pandas`).
Relative Path: A file address given relative to the folder your script is currently running in; more portable than an absolute path as it does not depend on a specific machine's folder structure.
Series: A 1-dimensional labelled array in Pandas, equivalent to a single column of a DataFrame; supports the same indexing and filtering operations as a NumPy array.
Third-party package: A package created by external developers that must be installed separately (typically using `pip`) before it can be imported; examples include `numpy`, `pandas`, and `matplotlib`.
Vectorisation: The application of a mathematical operation to every element of an array simultaneously, without writing an explicit loop; made possible by NumPy's use of optimised C code under the hood.
Virtual environment (venv): An isolated Python installation created for a specific project, allowing particular versions of packages to be installed without affecting other projects or the base Python installation.
