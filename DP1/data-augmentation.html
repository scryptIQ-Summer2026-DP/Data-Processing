    Columnar Operations

    Pandas DataFrames are mutable objects, which can have new columns of data added. The sole pre-requisite is that the newly added data must be the same length as the DataFrame, to ensure it matches, much like a NumPy concatenation.

    To create a new column we simply assign a new column name inside square brackets []; as though we were accessing a pre-existing column. A new data array is then assigned to it.

    However, we will seldom wish to add new data to a DataFrame. More often, we'll want to add a new data column based on, for instance, a combination of data columns, or a satisfied (or unsatisfied) logical condition. All of the mathematical operators +, -, /, *, … or logical operators <, >, ==, … will work row-wise when applied to one or more columns.

    Using the above is a quick and easy way to get statistical outputs per category, but is limited to that one statistic.

    If you would like to create a DataFrame containing several statistical measurements you can use the .agg() method (aggregate) method below. This method allows you to apply a function or list of functions to the columns (default) or rows (axis-1).

    It can look a bit intimidating at first, however, we will demonstrate it with one example, and then expand it to several.

    Here, instead of chaining a column with a method we use the aggregator method .agg() which contains:

    • the name of the new column

    • the column to apply the summary function to

    • the summary function

    These are within a dictionary where:

    • The new column name is the key

    • The value is a tuple of the column to apply to and its summary function

    The $**$ operator will unpack the dictionary so that it is in the proper format for .agg(). This can be a bit confusing, but it is a methodology worth committing to memory due to its usefulness.

    Altogether

    To bring everything covered in this notebook together, let's use .groupby() and .agg() to calculate some summary statistics on a subset of the penguin dataset, and then apply a formula to the result.

    In this example we will filter for female penguins, group by species and island, to then extract the mean, standard deviation, and count of bill length. These three values are all we need to calculate the 95% confidence interval around the mean for each group:

    From these metrics the 95% confidence interval around the mean can be calculated using this formula:

    $$CI_{95} = \frac{1.96 \cdot \sigma}{\sqrt{n}}$$

    where:

    • σ = the standard deviation

    • n = the number of samples

    • 1.96 = 95% of the distribution

    This could then be used for plotting or statistical analysis, something that will be explored further in the next lesson DP2.
