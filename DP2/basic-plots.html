    Introduction

    In the previous section we looked at the outliers on a subset of the data. However, it is hard to know whether you should be excluding the outliers you identify when you do not know the distribution of said data. Knowing the shape and spread of individual features in a dataset is a fantastic way to gain an understanding of what you have measured. For example, knowing a dataset has a bell-shaped curve can inform your choice of automatic outlier removal and statistical analysis.

    With Pandas and two plotting packages matplotlib and seaborn you can generate high-quality visualisations that can both inform your data exploration as well as create your final figures for the write-up.

    with Pandas

    The easiest ways to plot data from a Pandas DataFrame is using their in-built plotting methods. These can be used to create boxplots, line plots, scatter plots, histograms, and so on.

    Matplotlib

    Matplotlib is a comprehensive package for building single and multiple series of plots. The package gives you complete control of how the figure is altered, at the expense of having to write a little bit more code. In the final section we will introduce a further package seaborn that, like Pandas plots, is built upon matplotlib; allowing you to produce very visually appealing plots in one line. However, to interact with the figures post creation you will need to know how the matplotlib syntax works, so it is worth dedicating time to the older package!

    Like NumPy and Pandas, matplotlib is a third-party package that needs to be downloaded into your environment. Once that is achieved, we import the matplotlib pyplot submodule with the alias plt:

    The first action with matplotlib is to create an empty figure and axis to add your plots to. This is done through the .subplots() function. From there you can add plots to a given axes. Matplotlib was made to work with array like data, such as a Pandas series, a NumPy array, or a standard list.

    KEY TERMS

    figure: The top-level container for a matplotlib plot. A figure holds one or more axes and controls overall properties such as size and resolution.

    axes: The actual plotting area within a figure, where the data is drawn. A single figure can contain multiple axes (e.g. subplots).

    WARNING

    The Matplotlib boxplot function will return a blank plot if the data given to it contains any NaN values!

    The matplotlib figure acts as a blank canvas, upon which you layer more and more parts to build up the plot that you desire. The most basic is just the data on the y- or x-axis. Contextual information can be added through methods belonging to the axes object. A useful one is .set() which contains arguments that can produce the labels for:

    • the y-axis

    • the x-axis

    • the title

    NOTE

    The method xticklabels() lets you set the word under each tick, not the label of the x-axis. This can also be used to change the numbers for a continuous axis, such as ones seen in scatter plots. We will demonstrate this in the next section.

    The size of the figure is automatically calculated given the spread of the data present in the figure. However, it can be programmatically set with the argument figsize when passed to the .subplots() function.     Histograms

    Boxplots are useful tools for seeing the spread of the data, but they do not fully reveal its shape. A histogram fulfils this need by binning the values and counting how many fall into each bin, showing you the full distribution at a glance. Potentially highlighting any skew, normal distribution, or multiple peaks that would indicate distinct sub-populations.

    This matters because the shape of a distribution often drives the choices you make downstream. Many statistical techniques — including a number of machine learning (ML) methods — assume, or at least perform better on, features that approximate a normal distribution. Spotting skew, heavy tails, or bimodality early tells you whether a transformation (such as a log) might be warranted, or whether a feature is worth investigating more closely before modelling.

    To create a histogram, simply replace .boxplot() with .hist(). An additional argument that can be given to most matplotlib plots is color (colour), which can be passed the name of the colour as a string (see the list here) or a RGB tuple or hexcode (see here).

    Multiple sub plots

    Another benefit to using matplotlib directly is the creation of subplots - multiple plots on a single figure (canvas). This can allow you to directly compare and contrast different features and their sub-domains. To do so, all that needs to be done is to add an argument ncols to the plt.subplots() function, where the number (an integer) says how many plots on the horizontal axis there should be.

    Summary & Quiz

    Here we moved from summary numbers to visualising distributions. We began with the quick Pandas plotting methods, saw their convenience but limited control, then rebuilt the same plots in Matplotlib using the figure-and-axes model. From there we layered on labels, sizing, and multiple subplots. Histograms in particular revealed the shape of a feature — its skew, peaks, and tails — which sets up the transformations of the next section.
