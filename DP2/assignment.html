    Assignment DP2

    Assignment Background:

    This short assignment contains four questions that build upon and incorporate methods, techniques and concepts taught and demonstrated in the lesson materials. The questions will explore your understanding of data exploration, visualisation, transformation, and feature relationships.

    All questions aim to consider biological context. Timely submission will result in your assignment being returned within 14 days of the submission deadline. Marked Jupyter Notebook manuscripts will be returned with comments from your marker. If you receive a pass or distinction, your marker will attach the model solutions for this assignment via the open pull request on your lesson's GitHub repository.

    Assignment Questions:

    This assignment uses the obesity dataset, which can be found in the data folder of your DP2 lesson repository (obesity_dataset.csv). It contains 2,111 individuals from Mexico, Peru and Colombia, described by their eating habits and physical condition.

    It is recommended to use relative paths via the use of . or ..; for a reminder of their use please see the Pandas section of the online materials. If this is not working you can also obtain the absolute path in VSCode by right-clicking on the file inside the file explorer and selecting Copy Path.

    The columns you will be using are described below:

    Demographics / body measurements

    • Gender — Female / Male

    • Age — in years (continuous)

    • Height — in metres (continuous)

    • Weight — in kilograms (continuous)
    Eating habits
    • FCVC — frequency of consumption of vegetables (1 = never, 2 = sometimes, 3 = always)
    Any questions asking for your thoughts are purely for musings, you do not need to write an answer for marking!

    To confirm your code please print the result of each step, even when not explicitly asked

    Question 1

    In this question you will load the dataset, inspect it, reduce it to the columns of interest, and correct a data-type issue.

    Steps:

    1. Import pandas and load the dataset.

    2. View a statistical summary of all columns using .describe().

    3. Reduce the DataFrame so that it contains only the columns Age, Weight, Height, and FCVC.

    4. FCVC (frequency of vegetable consumption) is intended to be ordinal — a whole-number rating of 1, 2, or 3. Because ~77% of the dataset was generated synthetically with oversampling, it instead holds continuous values. Inspect this with .value_counts().

    5. Rectify the column by rounding its values to the nearest whole number with round(), reassigning the result back to the column. Confirm the change with .value_counts().

    Question 2

    You will now examine the distribution of Age, consider which transformation is warranted, and check for outliers.

    Steps:

    1. Plot a histogram of Age.

    2. The distribution is right-skewed. Given the knowledge of how age increase, would a log or a Yeo-Johnson transform be appropriate here? Based on your thoughts, transform the data with your decision.

    3. Using scipy.stats.zscore(), calculate the Z-scores of the raw and transformed Age values and identify any values with an absolute Z-score greater than 3.
    Should any of the values be removed if their scores were greater than 3?

    Question 3

    You will explore the relationships between the body-measurement features.

    Steps:

    1. Create a 2 column subplot with:
      • A scatter plot of Age (x-axis) against Weight (y-axis).

      • A scatter plot of Height (x-axis) against Weight (y-axis).

      • What relationship can you see for each? (No need to answer, just to muse)

    2. Confirm your intuition by calculating the Pearson correlation coefficient for each pair using .corr().

    Question 4

    Finally, you will investigate whether the Age–Weight relationship differs across levels of vegetable consumption (FCVC).

    Steps:

    1. Split the data into the three FCVC groups (1, 2, 3).

    2. Plot Age against Weight as a scatter, colouring each FCVC group differently on the same axes.

    3. Calculate the Pearson correlation coefficient between Age and Weight within each FCVC group.

    4. Present the three coefficients as a bar chart.
