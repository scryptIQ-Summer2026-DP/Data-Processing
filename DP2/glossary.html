Data Processing 2 - Glossary

Glossary DP2
Axes: The plotting area within a matplotlib figure, where the data is actually drawn. A single figure can hold several axes (for example, the panels of a subplot).
Bivariate: Involving two variables at once. Bivariate analysis looks at how two features relate to one another, for example through a scatter plot or a correlation coefficient.
Box-Cox transformation: A power transformation used to make data more normally distributed. It only accepts strictly positive values, and is generalised by the Yeo-Johnson transformation.
Box plot: A plot summarising the spread of a feature through its quartiles, showing the median, the interquartile range, and values that fall outside the whiskers as potential outliers.
Clipping: An outlier-handling strategy that replaces any value beyond a chosen threshold with the threshold value itself, retaining the row while limiting the influence of the extreme value.
Clustermap: An extension of the heatmap that hierarchically clusters rows and columns, grouping similar features together to reveal structure in the data.
Correlation coefficient (Pearson's r): A measure of the strength and direction of the *linear* relationship between two variables, always between −1 and +1. It is the covariance normalised by the product of the two standard deviations.
Correlation matrix: A symmetric table giving the correlation coefficient between every pair of features, with 1 along the diagonal (each feature correlated with itself).
Covariance: The raw, unnormalised measure of how two variables vary together. Unlike correlation it is unbounded, and its magnitude depends on the scale of the two variables.
Covariance matrix: A symmetric table of the pairwise covariances between features, with the variance of each feature along the diagonal.
Figure: The top-level container for a matplotlib plot. A figure holds one or more axes and controls overall properties such as size and resolution.
Heatmap: A plot that encodes the values of a matrix (such as a correlation matrix) as colours, making patterns across many features immediately visible.
Histogram: A plot that bins a feature's values and counts how many fall into each bin, revealing the shape of its distribution, including skew, peaks, and tails.
Hue: A seaborn argument that splits a plot by the categories of a chosen column, colouring each group separately without the need to filter the data manually.
Interquartile range (IQR): The difference between the 75th (Q3) and 25th (Q1) percentiles, spanning the middle 50% of the data. Values beyond 1.5 × IQR from either quartile are conventionally flagged as potential outliers.
Kernel Density Estimate (KDE): A smoothed, continuous curve fitted to data to estimate its probability distribution, offering an alternative to the binned histogram.
Logarithmic: Describes a relationship where values change by *multiplication* rather than *addition*, so each equal step represents a constant fold-change (e.g. ×10) rather than a constant amount.
Log transformation: Applying a logarithm to every value in a feature. It stabilises variance and pulls a right-skewed, multiplicative distribution towards symmetry, while keeping values interpretable as fold-changes.
Magic command: A command specific to IPython (and therefore Jupyter notebooks) that provides special functionality, such as `%matplotlib inline` for rendering plots within the notebook.
Matplotlib: A comprehensive Python plotting package offering complete control over a figure, at the cost of a little more code. It underpins the plotting in both Pandas and seaborn.
Maximum likelihood: An iterative process that estimates the parameter value best fitting the data — for example, the λ that best normalises a feature in the Yeo-Johnson transformation.
Multivariate: Involving many variables at once, as opposed to univariate (one) or bivariate (two).
Normal distribution (Gaussian): A symmetric, single-peaked, bell-shaped distribution. A *standard* normal distribution is the special case with a mean of 0 and a standard deviation of 1.
Normalisation: Transforming a feature so that its distribution approaches normality, often as preparation for statistical tests or machine learning tools that assume or perform better on normal data.
Outlier: A value that lies suspiciously far from the bulk of a feature's data. It may reflect genuine variation or a measurement error, and should only be removed once understood.
Pairplot: A seaborn matrix of plots showing a scatter for every pair of numerical features, with a distribution (KDE) on the diagonal — a quick overview of how features are distributed and related.
Pandas: A Python package for tabular data, providing the DataFrame and Series structures along with methods for exploring, summarising, and plotting data.
Percentile: The value below which a given percentage of the data falls. The 25th, 50th (median), and 75th percentiles are reported by `.describe()`.
Principal Component Analysis (PCA): A dimensionality-reduction technique that relies on the covariance structure of the data. It performs better on standardised, roughly normal features.
Pseudocount: A small constant (commonly 1) added to data before a log transformation so that zeroes, which cannot be logged, are brought above zero.
Quartile: The three points (Q1, Q2, Q3) that divide ordered data into four equal parts, at the 25th, 50th, and 75th percentiles.
Scatter plot: A plot of one feature against another, with each point an observation, used to inspect the relationship between two variables.
Seaborn: A visualisation library built on top of matplotlib and integrated with Pandas, able to produce attractive statistical plots from a single line of code.
Skew: The asymmetry of a distribution. A right- (positively) skewed feature has a long tail towards high values, which can violate the assumptions of tools such as the Z-score.
Spearman's rank correlation: A correlation measure that works on the *ranks* of the values rather than the raw values, making it robust to outliers and valid for any monotonic (not just linear) relationship.
Standard deviation: A measure of how spread out a feature's values are around its mean; the square root of the variance.
Standardisation: Rescaling a feature to have a mean of 0 and a standard deviation of 1, placing every feature on a common scale so that none dominates a model simply because its raw values are larger.
Subplot: Multiple plots arranged on a single figure, created with the `ncols` and `nrows` arguments of `plt.subplots()`, allowing features to be compared side by side.
Univariate: Involving a single variable. Univariate analysis examines one feature on its own, for example its mean, spread, or distribution.
Variance: The average squared deviation of a feature's values from its mean, quantifying how much the data vary across the population.
Yeo-Johnson transformation: A power transformation that pulls a distribution towards normality without assuming an underlying scale, and which (unlike Box-Cox) handles zero and negative values.
Z-score: The number of standard deviations a value lies from its feature's mean. A threshold of ±3 is conventionally used to flag outliers, though this is unreliable on skewed data.
