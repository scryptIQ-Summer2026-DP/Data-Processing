    Introduction

    Matplotlib has been a staple of mathematical plotting in Python for many years. Its greatest strength, the level of control you as the user have, is also one of its weaknesses; as you have to write a few lines of code at minimum - even for a basic plot. Python, if anything, is a language of abstraction through its large swathe of third-party packages. And in this spirit a collection of people have built seaborn, a visualisation library built on top of matplotlib, that can create visually appealing plots from a single line of code.

    In this quick section we will demonstrate how to make the plots we created in the previous sections, to demonstrate its usefulness. This is a bit of bonus material, that will be expanded upon in DP4.

    seaborn Syntax

    seaborn is typically imported with the alias sns, and with it, it has many plotting functions that will create all the basic plots covered, plus many more! Head to their gallery here for some visual inspiration from their toolbox.

    As well as being built from matplotlib, seaborn is built to integrate seamlessly with pandas. The first argument you pass to a seaborn plotting function is a pandas dataframe, the function can then take further arguments for styling.

    Our dataset is very wide, in that it has many columns (features), which makes viewing every pairing long winded and potentially visually unappealing. seaborn solves this with the pairplot function, that will create a matrix of $N$ dimensions, where $N$ is the number of numerical columns. This matrix, like the correlation matrix, is symmetrical with each unique pair having a scatter plot. On the diagonal there is a KDE plot, as it is plotting just that feature. When combined with a hue the pair plot becomes the go to tool to get a quick overview of how your data is distributed and related.

    Summary

    As a bonus, we recreated our earlier plots in seaborn, a library built on Matplotlib that produces attractive statistical graphics from a single line. Its hue argument splits data by category automatically, saving the manual filtering we did before, while functions like pairplot, KDE plots, heatmaps, and clustermaps give quick, rich overviews of how features are distributed and related. Because seaborn sits on top of Matplotlib, all the customisation tools we already know still apply.
